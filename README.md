# kve
Kerovoid Engine (KVE), a small OpenGL/SDL based game framework.
This project is mainly for my personal education.

# Features
* Abstraction of bare drivers with a portable API
* Sprite batching
* Tilemaps, loaded from Tiled's TMX format (https://www.mapeditor.org/)
* Rendering to a framebuffer

# Dependencies
* GLEW (OpenGL wrapper)
* SDL (window management)
* glm (vector math)
* rapidxml (loading tilemaps)
